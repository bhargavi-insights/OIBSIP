# 🛍️ Exploratory Data Analysis (EDA) on Retail Sales Data

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a Retail Sales dataset to uncover customer purchasing behavior, sales trends, and product performance. Using Python and data visualization techniques, the project transforms raw transactional data into meaningful business insights that support data-driven decision-making.

---

## 🎯 Objectives

- Understand the dataset structure and quality.
- Clean and preprocess the data.
- Analyze customer demographics and purchasing behavior.
- Identify top-performing product categories.
- Examine sales trends over time.
- Explore relationships between different variables.
- Generate actionable business recommendations.

---

## 📂 Dataset Information

| Column | Description |
|---------|-------------|
| Transaction ID | Unique identifier for each transaction |
| Date | Purchase date |
| Customer ID | Unique customer identifier |
| Gender | Customer gender |
| Age | Customer age |
| Product Category | Purchased product category |
| Quantity | Number of products purchased |
| Price per Unit | Price of one unit |
| Total Amount | Total transaction value |

## 📈 Key Insights

- Clothing and Electronics generated the highest revenue.
- Most customers belong to the 25–45 age group.
- Female customers contributed slightly higher overall sales.
- Higher purchase quantities resulted in higher transaction values.
- Sales increased during specific months, indicating seasonal demand.
- Product performance varied across different customer demographics.
